# 文件管理系统 - LocalStorage版本

基于浏览器的文件管理系统，使用LocalStorage实现数据持久化。

## 🌐 在线访问
https://你的用户名.github.io/file-manager-system/

## 📋 功能特点
- ✅ 用户登录/注册系统
- ✅ 文件上传/下载/管理
- ✅ LocalStorage数据持久化
- ✅ 数据备份与恢复
- ✅ 响应式设计
- ✅ 操作日志记录
- ✅ 用户权限管理

## 👥 演示账号
1. **管理员**：admin / password123
2. **普通用户**：user1 / password123

## 💾 数据管理
- 所有数据存储在浏览器LocalStorage中
- 支持数据导出为JSON文件备份
- 支持从JSON文件恢复数据
- 存储限制：5MB

## ⚠️ 注意事项
1. 清除浏览器缓存会删除所有数据
2. 不同浏览器之间的数据不共享
3. 建议定期备份重要数据
4. 首次使用时会创建默认数据

## 🛠 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
- Font Awesome图标
